HTML simplifié

Questions ou commentaires?

But

Activer le mode HTML de base sur une page donnée pour simplifier la lisibilité de la page en exécutant le strict minimum de manipulations du DOM.

Comment implémenter

Pour activer le mode HTML simplifié, vous n'avez qu'à ajouter le paramètre ?wbdisable=true à votre URL. Par exemple :

<a href="https://canada.ca/fr?wbdisable=true">Basculer vers le mode HTML simplifié</a>

Pour désactiver le mode HTML simplifié, vous n'avez qu'à ajouter le paramètre ?wbdisable=false à votre URL. Par exemple :

<a href="https://canada.ca/fr?wbdisable=false">Basculer vers le mode HTML simplifié</a>

Plugiciels autorisés à contourner le mode HTML simplifié

Certains plugins WET-BOEW peuvent être exécutés normalement lors de l'affichage de la version HTML simplifié d'une page. Voici la liste :

  • Data Ajax
  • Data JSON
  • JSON Manager
  • Envoie via Ajax

Pour autoriser l'exécution de l'un de ces plugiciels en version HTML simplifié, vous n'avez qu'à rajouter la classe CSS .wb-disable-allow sur l'élément. Par exemple :

<div class="wb-disable-allow" data-wb-ajax='{ "url": "ajax/data-ajax-extra-fr.html" }'></div>
