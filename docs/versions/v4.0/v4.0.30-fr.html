v4.0.30.x - Notes de version

Version 4.0.30

  • Date de lancement : 2019-02-11
  • Téléchargements

Quoi de neuf?

Version 4.0.30

  • Correctif - Lightbox - Fermeture de la fenêtre superposé lorsqu'on clique sur un lien de la page courante.
  • Mineur - Tableaux - Interface de filtrage, ajout du support pour input[type=date] lors de la réinitialisation
  • Mineur - Fils de syndication - Ajout de la lecture directe des fichiers XML qui sont dans la limite des CORS.
  • Correctif - Menu - Ouverture du menu mobile lorsque le menu est statique dans la page
  • Mineur - Fils de syndication - Lecture d'un fil d'actualité en format JSON similaire à ATOM XML.
  • Mineur - Fils de syndication - Retrait de la lecture d'un fil d'actualité via l'API Yahoo. Ceci est remplacé par une lecture directe.
  • Mineur - Tableaux - Retrait de la pagination lorsqu'il y a seulement une seule page de résultat
  • Mineur - Tableaux - Interface de filtrage, ajout l'option de filtrer les résultats avec des case à cocher
  • Correctif - Chronologie intégée Twitter - Retrait de l'example avec code obsolete de Twitter.
  • Correctif - Tableaux - Interface de filtrage - adjustement lorsqu'il y a plusieur case à cocher pour plus qu'une colonne.
  • Développeur de la BOEW - Modification du script de compilation et de ses dépendances.
