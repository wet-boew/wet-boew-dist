Merge Server-Client Errors

  • Documentation
  • Questions or comments?
  • Form Validation
  • Merge Server-Client Errors

Purpose

This component merges Server and Client errors together.

Example

After submitting your page, the server returned 4 server errors. Now if you try to resubmit your form with some client errors, the component will add these errors to the existing server errors.

To test: Try clearing a field or enter 201 as the Beneficiary's age or uncheck all checkboxes.

Demo Asp.Net Web Form

The form could not be submitted because 4 errors were found.

  • Error 1: Beneficiary name - The name should begin with "Super" - (Correct and resubmit)
  • Error 2: Beneficiary age - Age should be between 64 and 200 for a Retirement benefit - (Correct and resubmit)
  • Error 3: Contact Information - You can't choose more than 2 checkboxes - (Correct and resubmit)
  • Error 4: I agree with the directives - This field is mandatory - (Correct and resubmit)
