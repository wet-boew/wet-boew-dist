Data Scrub

Questions ou commentaires?

But

Ce composant offre la possibilité de supprimer les informations personnelles identifiables (PII) dans les champs d'un formulaire avant de le soumettre.

Voici la liste des IPI que le plugiciel recherchera et supprimera:

  • Toutes les combinaisons de 8 chiffres et plus
  • Numéro de série du passeport canadien
  • E-mail
  • Code postal canadien
  • Toute valeur de nom d'utilisateur potentielle ajoutée dans ce format username=theUserValue, username:theUserValue, user=theUserValue, user:theUserValue
  • Toute valeur de mot de passe potentielle ajoutée dans ce format password=thePassValue, mot de passe:thePassValue pass=thePassValue, pass:thePassValue

Pour plus de détails sur le comportement de nettoyage des PII, veuillez consulter l'assistant de base wet-boew:wb.findPotentialPII

Utiliser lorsque

  • les formulaires peuvent contenir des informations personnelles identifiables (PII), ce plugiciel identifiera et supprimera ces informations avant la soumission du formulaire.

Ne pas utiliser lorsque

  • un formulaire utilise le plugiciel Validation de formulaires
  • un formulaire utilise le plugiciel Envoie via Ajax

Comment implémenter le plugiciel

  1. Ajoutez l'attribut de données suivant data-rule-wbscrub pour chaque élément du formulaire susceptible de contenir des informations personnelles. 
    <input id="email" name="email" type="email" data-rule-wbscrub />
  2. Optionnellement, si vous souhaitez que le formulaire soumette un paramètre chaque fois qu'un PII est trouvé et supprimé, créer un champs de type masquée avec data-wbscrub-flag avec une valeur et un nom de votre choix 
    <input name="scrub" value="1" data-wbscrub-flag />

Événements

Documentez les événements publics qui peuvent être utilisés par les implémenteurs ou les développeurs.

Événement Déclencheur Effet déclenché
wb-init.wb-data-scrub Déclenché manuellement (e.g., $( ".wb-data-scrub" ).trigger( "wb-init.wb-data-scrub" );). Utilisé pour initialisé manuellement le plugiciel Supprime-IPI.
wb-ready.wb-data-scrub (v4.0.24+) Déclenché automatiquement après l'initialisation du plugiciel Supprime-IPI. Utilisé afin d'identifié l'élément sur lequel le plugiciel Supprime-IPI a été initialisé (la cible de l'événement) 
$( document ).on( "wb-ready.wb-data-scrub", ".wb-data-scrub", function( event ) {});
$( ".wb-data-scrub" ).on( "wb-ready.wb-data-scrub", function( event ) {});
wb-ready.wb (v4.0.24+) Déclenché automatiquement lorsque la WET-BOEW a terminé son chargement et son exécution. Utilisé pour identifié le moment lorsque tous les plugiciels et tous les polyfills de la WET-BOEW ont terminés de charger et de s'exécuter. 
$( document ).on( "wb-ready.wb", function( event ) {});
