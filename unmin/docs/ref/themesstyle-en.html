Themes and style

Themes

  • Canada.ca theme - This theme implements the layout and design requirements for Canada.ca.
  • Web Experience Toolkit (WET) theme - This theme provides an example implementation of the Base Theme.

Abandoned themes

The following themes was last released with WET version 4.0.27.

  • Base theme - A theme template to aid in the creation of new themes.
  • Government of Canada Web Usability Theme - This theme implements the requirements of the Standard on Web Usability (under review) and has been replaced by the Canada.ca theme (above).
  • Government of Canada Intranet theme - This theme adapts the Government of Canada Web Usability theme for intranet sites.
  • Open Government Platform (OGPL) theme - The Open Government Platform (OGPL) theme was developed to support the Open Government Platform (OGPL).
