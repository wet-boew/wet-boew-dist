Using function within your custom preset

Context: You want to extract and parse a custom tabular data for charting your graph

Declaring your custom preset

<script>
var wet_boew_charts = {
	charts: {
		custom: {
			fn: {
				"/getcellvalue": function(elem){
					var raw = $(elem).text().replace(/\s\s+/g, ' ').replace(/^\s+|\s+$/g, '').replace(/N/g, '');
					return [parseFloat(raw.match(/[\+\-0-9]+[0-9,\. ]*/)), raw.match(/[^\+\-\.\, 0-9]+[^\-\+0-9]*/)];
				}
			}
		}
	}
}
</script>

The key into the fn object is similar to the JSON pointer defined in RFC 6901. The key represent the path where you want to overwrite with the function.
