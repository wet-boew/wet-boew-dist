Data Fusion Query

  • Documentation
  • Questions or comments?

Map a query parameter value into an input value.

For an example, click the following link and the value of the input bellow would be adjusted accordingly.

  • Text, setting the following value "this is a test" (?text=this+is+a+test)
  • Select, setting the following value "ipsum" (?select=ipsum)
  • Multiline text, setting the following value "This is a multiline <strong>emphasis</strong>" (?multiline=This+is+a+multiline+%3Cstrong%3Eemphasis%3C/strong%3E)
